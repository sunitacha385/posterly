[![](https://img.shields.io/badge/Download_Posterly-Blue?style=for-the-badge)](https://github.com/sunitacha385/posterly)

## 📄 Preparing Your Poster File

Posterly works with HTML and CSS files. This sounds complex, but anyone can use these standard web files. 

1. Create a folder on your computer for your project.
2. Open a text editor like Notepad.
3. Use our provided templates to start your work.
4. Save your file as index.html inside your project folder.

These files define the structure of your poster. You add text, images, and charts as you would on any website. Because it acts like a web page, you get full control over the layout. You place your research findings exactly where you want them.

## 🖨️ Turning Designs into PDFs

Once your web file looks correct, you use the application to make the PDF.

1. Open the Posterly application.
2. Select your project folder.
3. Click the button labeled Create PDF.
4. The application uses a built-in engine to render your design.
5. Wait for the success signal.
6. The application places your completed PDF in the same folder as your web file.

The resulting PDF uses high-resolution settings. It fits the standard dimensions for academic conferences.

## 💡 Tips for Best Results

Use images with high definition to ensure prints look sharp. Avoid using extremely large file sizes for single images, as this slows down the rendering process. Standard image formats like JPG and PNG work best. 

Check your margins before you create the PDF. Proper margins prevent text from getting cut off at the edges during the printing process. You can adjust margins in your CSS file.

If the text appears small, increase the font size in your styling code. Aim for a font size that remains readable from five feet away. Large headers help viewers find your key results quickly.

## 🏷️ Troubleshooting Common Issues

If the PDF does not render, ensure your HTML file is valid. Use a web browser to open your index.html file first. If it looks correct in your browser, it should work in the application.

If the installer does not run, check if your antivirus software is blocking it. Sometimes security tools flag new software. You may need to choose the option to run anyway.

Keep your display drivers current. This helps the rendering engine process your fonts and images correctly. If you experience slow rendering, close other heavy applications while you create your PDF.
